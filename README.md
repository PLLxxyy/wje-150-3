# 图书漂流 - 让书本旅行

一个纯前端的图书漂流换书平台，所有数据存储在浏览器 localStorage 中，无需后端服务。

## 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **路由**: React Router 6
- **数据持久化**: 浏览器 localStorage
- **无后端 / 无数据库**

## 启动方式

```bash
npm install
npm run dev
```

启动后访问 http://localhost:5173 即可使用。首次加载会自动注入 mock 示例数据。

## 功能说明

### 首页 - 书籍列表
- 顶部搜索框，按书名或作者实时搜索
- 分类筛选：文学 / 科技 / 生活 / 童书 / 社科 / 艺术 / 经管 / 其他
- 状态筛选：待漂流 / 漂流中 / 已归档
- 点击书籍卡片进入详情页

### 发布书籍
- 填写书名、作者、分类、简介，可选上传封面图
- 选择发布方式：**漂流**（寄出后下一个人继续传）或 **换书**（两人互换）
- 发布后自动生成专属漂流码

### 书籍详情
- 展示书籍完整信息与当前状态
- 专属漂流码展示，可分享给他人查看漂流轨迹
- 漂流时间线：记录每一位持有者的操作（发布 / 寄出 / 收到 / 完成）
- 足迹地图：展示这本书去过的城市
- 寄出 / 收到 / 归档操作按钮

### 个人中心
- **我发布的**：查看自己发布的所有书籍
- **我手中的**：当前持有的书籍
- **漂流历史**：参与过的所有漂流记录
